# PHP Marketplace Web Application

## Overview
This project is a **PHP-based Marketplace web application** that allows users to browse products, register and log in, search items, view offers, and interact with marketplace features.  
It demonstrates core web development concepts using **PHP, HTML, CSS, JavaScript**, and a **hosted database environment**.

The application was deployed on a web hosting platform that provided database support, enabling persistent storage of user and application data.

---

## Features
- User registration, login, and logout
- Session-based authentication
- Product browsing and category-wise product pages
- Search functionality
- Recently viewed products
- Top products and offers display
- User ratings and feedback submission
- Static informational pages (About Us, Contacts, News)

---

## Technologies Used
- **PHP** – Server-side logic
- **HTML & CSS** – UI structure and styling
- **JavaScript** – Basic client-side interactions
- **Hosted Database** – Provided by the web hosting platform
- **Apache Web Server** – Via hosting provider

---

## Database & Deployment
The application was hosted on a web hosting platform that included an integrated database service.  
All user and application data were stored and managed through this hosted database during deployment.
