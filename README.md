PHP Marketplace Web Application
Overview:

This project is a PHP-based Marketplace web application that allows users to browse products, register and log in, search items, view offers, and interact with marketplace features. The application demonstrates core web development concepts using PHP, HTML, CSS, and a hosted database environment.

The project was deployed on a web hosting platform that provided database support, enabling persistent storage of user and application data during deployment.

Features:

User registration, login, and logout

Session-based authentication

Product browsing and category-wise product pages

Search functionality

Recently viewed products

Top products and offers display

User ratings and feedback submission

Static informational pages (About Us, Contacts, News)

Technologies Used:

PHP – Server-side logic

HTML & CSS – UI structure and styling

JavaScript – Basic client-side interactions

Hosted Database – Data storage provided by the web hosting platform

Project Structure:

Marketplace/

│── login.php / register.php # User authentication
│── logout.php # Logout handling

│── config.php # Application configuration
